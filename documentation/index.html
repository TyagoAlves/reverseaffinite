1 Início Rápido

Em 3 passos você tem o reverseaffinite rodando no Linux:

1

Clone e instale dependências

$ git clone https://github.com/TyagoAlves/reverseaffinite.git
$ cd reverseaffinite
$ pip install PyQt5 numpy
2

Inicie o editor

$ python3 main.py

A janela do editor abrirá com layout profissional: paleta de ferramentas à esquerda, canvas central, painéis à direita.

3

Crie ou abra uma imagem

Arquivo → Novo (Ctrl+N) para criar uma tela em branco ou Arquivo → Abrir (Ctrl+O) para editar uma imagem existente. Formatos: PNG, JPG, BMP, TIFF, GIF.

T Ferramentas

15+ ferramentas profissionais organizadas na paleta vertical à esquerda. Cada ferramenta tem atalho de tecla única — como no Photoshop.

V Move

Arrasta camadas e seleções. Shift mantém proporção. Nós nos cantos para redimensionar/rotacionar.

M Seleção Retangular

Seleções retangulares. Shift = quadrado perfeito.

L Lasso

Seleção à mão livre. Ideal para contornos orgânicos.

W Varinha Mágica

Seleciona por similaridade de cor. Tolerância ajustável.

B Brush

Pincel com tamanho, opacidade, flow. Suporta tablet com pressão.

E Eraser

Apaga pixels da camada ativa. Funciona em máscaras.

P Pen

Curvas Bezier com handles. Adiciona/remove âncoras. Path → Selection.

G Gradient

Múltiplos stops, tipos linear/radial/angular/reflected.

S Shape

Retângulos, elipses, triângulos, polígonos. Fill sólido ou gradiente.

T Text

Texto com fonte, tamanho, cor e alinhamento. Edição inline.

C Clone Stamp

Clona pixels. Alt+clique define origem.

H Healing Brush

Corrige imperfeições combinando textura e iluminação.

O Crop

Corta imagem. Preview com grade de terços.

I Color Picker

Seleciona cores do canvas. Define foreground/background.

Z Zoom

Clique para ampliar, Alt+clique para reduzir.

L Camadas & Mesclagem

Sistema completo com 12 modos de mesclagem, máscaras, grupos e camadas de ajuste não-destrutivas.

Painel de Camadas

Listagem: Thumbnail, nome, visibilidade (olho), opacidade e modo de mesclagem.

Ordem: Arraste para reordenar. Camadas no topo aparecem na frente.

Grupos: Pastas que agrupam camadas com opacidade/mesclagem aplicada ao grupo.

Máscaras: Preto oculta, branco revela. Edição não-destrutiva.

Lock: Trave para evitar edição acidental.

Modos de Mesclagem (12)

Normal

Camada cobre a de baixo. Opacidade controla transparência.

Multiply

Escurece combinando cores. Branco = sem efeito.

Screen

Clareia a imagem. Preto = sem efeito.

Overlay

Combina Multiply e Screen. Aumenta contraste.

Darken

Mantém pixels mais escuros de cada canal.

Lighten

Mantém pixels mais claros de cada canal.

Difference

Subtrai cores. Branco = inverte.

Add

Soma canais. Efeito de luz/brilho.

Subtract

Subtrai canais. Escurece a imagem.

Dodge

Clareia baseado no tom da camada.

Burn

Escurece baseado no tom da camada.

Soft Light

Similar ao Overlay, mais suave. Bom para texturas.

Filtros Destrutivos

Camadas de Ajuste (Não-destrutivas)

→Hue/Saturation: Matiz, saturação e luminosidade.

→Levels: Controle de tons com histograma.

→Curves: Curvas de tom com pontos de controle.

Pen & Paths

Ferramenta Pen com curvas Bezier cúbicas — como no Illustrator ou Affinity Designer.

Como Usar

1. Selecione Pen (P)

2. Clique para adicionar âncoras

3. Arraste ao clicar para criar handles Bezier

4. Clique no primeiro ponto para fechar o path

5. Clique direito em um ponto para removê-lo

Ações

Converter em Seleção: Ctrl+Enter ou botão direito

Preencher Path: Com a cor de frente

Contornar Path: Com o brush atual

Arquitetura do Projeto

O reverseaffinite é modular: engine Python para prototipação, engine C++ para produção.

Estrutura de Diretórios

reverseaffinite/
├── main.py                    # Ponto de entrada
├── editor/
│   ├── app_ui.py              # MainWindow, menus, layout
│   ├── canvas.py              # CanvasView, renderização
│   ├── tools.py               # Sistema de ferramentas
│   ├── layers.py              # Gerenciamento de camadas
│   ├── filters.py             # Filtros de imagem
│   ├── history.py             # Undo/redo c/ snapshots
│   ├── panels.py              # Painéis laterais
│   ├── brushengine.py         # Motor de pincéis
│   ├── gradient.py            # Gradientes
│   ├── gradient_editor.py     # Editor visual gradiente
│   ├── path.py                # Curvas Bezier
│   ├── snapping.py            # Sistema de snapping
│   ├── guides.py              # Guias/réguas
│   ├── i18n.py                # Internacionalização
│   ├── settings.py            # Configurações
│   └── resources.py           # Tema escuro
├── cpp_editor/                # Engine C++
│   ├── CMakeLists.txt
│   ├── src/                   # Código fonte
│   └── include/               # Headers
├── locale/                    # Arquivos de tradução
├── tests/                     # 136+ testes
└── tools/                     # Utilitários

Engine C++

Motor de renderização nativo com Qt5 e OpenGL para performance de produção.

⚡ OpenGL

  • Canvas acelerado por GPU
  • Zoom/pan suaves a 60fps
  • Blend modes em GLSL shaders
  • Framebuffer objects offscreen

⚡ SIMD

  • SSE2 128-bit em pixels RGBA
  • Blend modes sem branch
  • Fallback escalar
  • Binário ~104 KB

Build

$ cd cpp_editor && ./build.sh

Requer Qt5 (aqtinstall) e CMake.

FAQ

Dúvidas frequentes sobre o reverseaffinite.

Funciona no Windows ou macOS?

O foco é Linux, mas como usa Python/PyQt5, funciona em Windows e macOS com pequenos ajustes.

Preciso de placa de vídeo?

Não. O engine Python roda em CPU (NumPy). O C++ usa GPU se disponível, com fallback CPU.

Tem suporte a plugins?

Ainda não, mas a arquitetura de ferramentas é extensível por herança da classe Tool. Futuramente teremos plugins Python.

Diferença para o GIMP?

O reverseaffinite busca o fluxo do Affinity Photo / Photoshop — layout profissional, paleta à esquerda, painéis à direita, atalhos de tecla única. Mais leve e moderno.

Como contribuir?

Faça fork no GitHub, implemente e abra um Pull Request. O projeto é 100% open source (MIT).
